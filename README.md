# README

## About

Display the position of political actors.

## How to run

Run locally on port 4321 (localhost:4321) with:

```
$ pnpm run dev
```

Build static files:

```
$ pnpm run build
```

Output in `./dist`. Open `./dist/index.html` on your preferred browser.
